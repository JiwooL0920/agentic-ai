# Agentic AI Frontend

Next.js frontend for the Agentic AI Platform.

## Features

- **Next.js 14 App Router**: Modern React framework with server components
- **shadcn/ui**: Beautiful, accessible UI components
- **Tailwind CSS**: Utility-first styling
- **Blueprint Selector**: Choose from multiple agent blueprints
- **Streaming Chat**: Real-time SSE streaming from backend

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```
