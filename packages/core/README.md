# Agentic AI Core

Python core library for the Agentic AI Platform.

## Features

- **OllamaAgent**: Local LLM agent implementation using Ollama AsyncClient
- **SupervisorOrchestrator**: Multi-agent routing and coordination
- **FastAPI Backend**: RESTful API with streaming support
- **Agent Factory**: Load agents from YAML configurations
- **Storage Adapters**: DynamoDB, S3, Redis, PostgreSQL+pgvector support

## Installation

```bash
pip install -e ".[dev]"
```
